## ğŸ¯ Overview

**WeRun 2.0** is a comprehensive running tracker application built with Flutter and Firebase. It provides real-time GPS tracking, activity history, and user authentication, helping runners track their progress and achieve their fitness goals.

The application follows clean architecture principles with clear separation between presentation, domain, and data layers, making it maintainable and scalable.

---

## âœ¨ Features

### Core Features
- **Real-time GPS Tracking**: Track your running routes with accurate GPS positioning
- **Activity History**: View and analyze your past running sessions
- **Distance Calculation**: Automatically calculate distance traveled using the Haversine formula
- **User Authentication**: Secure login and registration with Firebase Auth
- **Profile Management**: Manage user profiles and preferences
- **Cross-Platform**: Runs on Android, iOS, Web, Windows, macOS, and Linux

### Technical Features
- Clean Architecture implementation with BLoC pattern
- Dependency Injection using GetIt and Injectable
- Firebase integration (Auth, Firestore)
- Custom theming and styling
- Responsive UI design

## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Flutter SDK**: >= 3.0.0
- **Dart SDK**: >= 3.0.0
- **Android Studio** or **Xcode** (for mobile development)
- **Firebase Account**: For backend services
- **Git**: For version control

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/yourusername/dacs4_werun_2_0.git
cd dacs4_werun_2_0
```

2. **Install Flutter dependencies**

```bash
flutter pub get
```

3. **Generate code for dependency injection**

```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

### Configuration

1. **Firebase Setup**

   - Create a new Firebase project at [Firebase Console](https://console.firebase.google.com)
   - Add your Android/iOS apps to the Firebase project
   - Download `google-services.json` (Android) and `GoogleService-Info.plist` (iOS)
   - Place them in the appropriate directories:
     - Android: `android/app/google-services.json`
     - iOS: `ios/Runner/GoogleService-Info.plist`

2. **Firebase Configuration**

   Update `lib/firebase_options.dart` with your Firebase project credentials:

```dart
// This file is auto-generated by FlutterFire CLI
// Run: flutterfire configure
```

3. **Enable Firebase Services**
   - Firebase Authentication (Email/Password)
   - Cloud Firestore
   - Firebase Storage (if needed)
